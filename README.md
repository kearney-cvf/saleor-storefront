# saleor-storefront

- Saleor store front

## Getting Started

### Prerequisites

- Node.js >=12 <14
- Yarn

### Run

1. Clone the repository:
<pre>
$ git clone https://github.com/kearney-cvf/saleor-storefront.git
</pre>

2. Go to the cloned directory:
<pre>
$ cd saleor-storefront
</pre>

3. Install packages:
<pre>
$ yarn
</pre>

4. Run the application dev mode:
<pre>
$ yarn start
</pre>

### Test

<pre>
$ yarn test
</pre>

### Run storybook

<pre>
yarn storybook
</pre>
